# Datos

No se incluyen datos originales en este repositorio para evitar exponer información sensible o archivos académicos no autorizados.

## Recomendación

Para hacer el repositorio más profesional, se puede agregar:

1. Una muestra pequeña de datos ficticios.
2. Datos anonimizados.
3. Un archivo con la estructura esperada de las tablas.
4. Una explicación de cómo cargar los datos localmente.

## Carpetas sugeridas que no se subirán por defecto

- `data/raw/`
- `data/original/`

Estas carpetas están ignoradas en `.gitignore`.
