# Power BI

En esta carpeta se puede incluir el archivo `.pbix` del dashboard y capturas exportadas para mostrar el resultado visual del proyecto.

## Recomendaciones

1. Guardar aquí el archivo `dashboard-cartera.pbix` si no contiene información sensible.
2. Exportar capturas en formato `.png` de las páginas principales del dashboard.
3. Colocar las capturas dentro de `powerbi/capturas/`.
